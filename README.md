## ⚙️ How to Use Nameplate

Nameplate runs quietly in the background after you start it. You configure your settings through the system tray area located at the bottom right of your taskbar.

### Setting Your Labels
Open the settings menu from the tray icon. You can type the name of the computer in the text field. This name appears as an overlay on your desktop. You may adjust the position of this label to avoid covering desktop icons.

### Adjusting Colors
Select the frame color from the color picker. A bright color helps you distinguish between production machines and test machines. Keep the contrast high so the name remains readable against your wallpaper.

### Adding Watermarks
You can upload an image file to function as a watermark. The software places this image over your desktop background. Use this for company logos or security notices. We recommend using a transparent file for the best visual results.

### Using the Click-Through Feature
The software allows your mouse cursor to pass through the overlay. This means you can still click icons and files on your desktop without moving the Nameplate label. You stay productive while maintaining visual awareness of your device.

## 🆘 Troubleshooting Common Issues

Most users find the installation process simple. If you encounter issues, please review these common solutions:

*   **Installer does not start:** Check your browser settings to ensure it permits the download of executable files. If you use antivirus software, it may flag the file. Please add an exception for Nameplate to allow it to run.
*   **Overlay does not appear:** Ensure the background application icon displays in your system tray. Right-click the icon and choose "Show Labels" to toggle the display.
*   **Settings do not save:** Ensure you have write permission for the installation folder. If you installed the app in a restricted system directory, try running the application as an administrator.
*   **Labels look blurry:** High-resolution monitors sometimes struggle with overlay scaling. Go to the settings menu and increase the font size or switch to a higher resolution image file for your watermark.

## 📝 Frequently Asked Questions

**Does the software slow down my computer?**
No. Nameplate uses very little memory or processing power. It stays dormant until you change your desktop background or adjust a setting.

**Can I use this on a laptop?**
Yes. The software works perfectly on laptops, workstations, and thin-client devices.

**Does it change my actual desktop wallpaper?**
The software does not alter your original image file. It creates a secondary layer that floats on top of your wallpaper. You can turn this layer off at any time without losing your original desktop look.

**Will the labels show up in my screenshots?**
Yes. The labels are part of the desktop view. They appear in any screenshot you take while the software is active. This feature helps document technical issues for support tickets.
